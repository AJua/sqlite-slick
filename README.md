# SQLite Slick

[SQLite](http://www.sqlite.org/) is a self-contained and serverless SQL database engine. I've put together an example of how to use it in scala using [Slick](http://slick.typesafe.com/).

## Technology
+ SBT 0.13
+ Scala 2.10.3
+ Java 1.7.0_25

## Run
With SBT on the path:

```
sbt run
```
